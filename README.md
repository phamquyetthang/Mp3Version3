**Cấu trúc thư mục**
```
src
├── components
├── styles
├── containers
├── screens
│   ├── Dashboard
│   └── etc ...
├── saga
└── index.js
```
**Chạy trên ubuntu:**
- `cd android/`
- `chmod +x gradlew`
- Hoặc `sudo chmod 777 gradlew`
  
( lỗi do không có quyền chuy cập file gradlew )

**Chạy thử trên Android** 
- Vào folder : `/product_build/`
- Tải file apk
